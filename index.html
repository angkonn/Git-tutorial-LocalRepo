Terminal basics: #To change directory in terminal: 'cd directory name' #To get out of a directory: 'cd ..' #To create a new directory: 'mkdir directory name' Git: # Set name: git config --global user.name "Name" # Set email git config --global user.email "Email" # Clone repo git clone # Stage file git add # Stage all files git add . # Commit changes git commit -m "Message" # Show status git status # Show commits git log # Push changes git push origin main Init Command: init- used to create a new git repo git init git remote add origin <- link -> git remote -v (to verify remote) git branch (to check branch) git branch -M main (to rename branch) git push origin main git push-u origin main (setting upstream)
